# Contributing

Thank you for improving CVD.AI Mobile.

## Workflow

1. Create a branch for your change.
2. Keep changes focused and easy to review.
3. Run formatting, analysis, and tests before opening a pull request.

```bash
dart format lib test
flutter analyze
flutter test
```

## Pull Request Checklist

- The app still runs.
- `flutter analyze` passes.
- `flutter test` passes.
- Medical or risk-related text is careful and does not claim diagnosis.
- Screens remain mobile-friendly.
