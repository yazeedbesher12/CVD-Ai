# CVD.AI Mobile

CVD.AI Mobile is a Flutter demo app for AI-assisted ECG risk screening. It guides a user through profile details, medical risk factors, symptoms, a simulated real-time ECG reading, and a final risk result with an exportable PDF report.

> Important: This project is a demonstration and screening-support prototype. It is not a medical diagnosis tool and must not replace professional medical evaluation.

## Features

- User and admin entry flows
- Medical risk factor and symptom scoring
- Simulated ECG generation based on normal, irregular, and high-risk patterns
- Animated real-time ECG preview
- Weighted final risk score using ECG and profile data
- Mobile-style app frame for desktop/web demos
- PDF report export with patient details, ECG image, clinical summary, and recommendation

## Tech Stack

- Flutter / Dart
- `pdf` for report generation
- `printing` for PDF export and sharing

## Getting Started

### Requirements

- Flutter SDK matching the project SDK constraints
- Dart SDK included with Flutter
- A device, emulator, browser, or web-server target

### Install Dependencies

```bash
flutter pub get
```

### Run

Mobile or desktop:

```bash
flutter run
```

Web server demo:

```bash
flutter run -d web-server --web-hostname 0.0.0.0 --web-port 5000
```

On Windows, the `printing` plugin may require Developer Mode because Flutter plugins use symlinks.

## Medical Disclaimer

The ECG data in this app is simulated for demonstration purposes. The app does not connect to a smartwatch or medical-grade ECG sensor. Any report generated by the app is for prototype demonstration only.
